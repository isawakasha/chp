Calculates hiddenpower for the league17.ru in the terminal

# Compiling

```bash
cargo build --release
```

# How to use

```bash
chp <gens>
```

## Example

```bash
chp h35a35d35s35sa35sd35 h3a18d25s15sa4sd6
```

Output

```
h35a35d35s35sa35sd35 Steel
h3a18d25s15sa4sd6 Electric
```
